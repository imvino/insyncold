Add a User

      

To create a new user account:

1.Click Edit Users from the Account menu. The Edit Users screen displays.
2.Click New User. The Add New User dialog box displays.
3.Type a name for the user account in the User Name field.
4.Type a unique password in the Password field.
5.Re-type the password in the Confirm Password field.
6.Select which user permissions should be assigned to the user. Refer to Managing Users for what functionality is accessible from each permission level.
7.Select the maximum amount of time that you want to elapse before the In|Sync WebUI automatically logs the user out from the Session Timeout drop-down list. The default value is 30 minutes.
8.Click Save.

8.Click Propagate Users to save the user to all the processors on the management group.
