Confirm correct InSync operation

After the inputs and outputs have all been set up, you need to confirm that InSync is working correctly. To verify that InSync is configured correctly/functioning normally:

1.

Click InSync from the Advanced screen. The InSync dialog box displays.

2.

Observe the icon view of the intersection in the top center, verifying that it reflects the correct number of lanes for each phase assigned.

3.

Click the I/O Board tab. Verify that Communication Statuses all read OK.
