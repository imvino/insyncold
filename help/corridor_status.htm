Status

      

The Status page provides a real-time status of communications between In|Sync Processors within the management group, allowing you to troubleshoot network problems. It provides the following information:

•Whether or not a processor is online or offline
•If a processor is online, but unable to physically reach certain processors.

 

For instance, if a management group contains three intersections — A, B, and C — and C is down, A and B will individually report that they are online but unable to talk with C. Once all communications are restored, this page will indicate that it is waiting for the processors to validate communications.
