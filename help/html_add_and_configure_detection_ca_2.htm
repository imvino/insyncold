Add Detection Cameras

The following instructions pertain only to InSync and InSync:Fusion installations.

To define that an InSync detection camera is located on an approach:

1.

Click the Direction drop-down box, select the direction you want the camera on, and click  on the camera tab.

2.

Type the IP address associated with the InSync detection camera in the IP field and click OK.

3.

Continue configuring the InSync detection cameras by creating detection zones, fog zones, presence segments, etc.

4.

Go to the All Direction view to assign the cameras to a lane(s). A single camera detection zone may only be assigned to one lane.

5.

To assign a camera detection zone to a lane, right-click the lane in the All Direction view and select Add Detector. Select the detector that you want associated with that lane.

NOTE:

Adding a second camera to a direction will create an Advanced Camera Detector. An unlimited number of advanced cameras can be added per direction. After adding advanced camera detection zones, be sure to assign them to the lanes in the All Directions view.
