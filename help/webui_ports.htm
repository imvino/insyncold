WebUI Ports

      

The following ports must be open to use the WebUI:

▪HTTP access (port 80) and/or HTTPS access (port 443) to each In|Sync network device (In|Sync Processor, monitoring camera, DIN Relay, and Ethernet repeater) and the WebUI
▪RDP access (port 3389) to each In|Sync Processor
▪SMTP access (port 25) for outbound access to Rhythm mail server at 98.100.100.2
▪Port 20000 and 445 for In|Sync communications
▪554: RTSP port
▪4520: Device control and streaming for ActiveX Control
▪943: Silverlight policy
▪Port 123 for NTP (outbound to NTP server 74.222.222.166)

If you cannot access the WebUI, check with your network administrator to be sure these ports are open.
