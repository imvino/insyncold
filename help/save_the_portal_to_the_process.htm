Save the Portal to the Processor

      

After defining the content that you want displayed in the portal and arranging the content to your liking, you must then save the view so that it can be accessed for later viewing. To do this, click the Save to Processor button. This allows you to obtain this portal whenever you connect to that specific processor. This feature is useful when and if you are not finished designing the portal, but you need to step away and come back to finish the design.
