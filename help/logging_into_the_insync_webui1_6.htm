Logging into the In|Sync WebUI

      
1.Open your preferred web browser and type the processor's IP address in the Address bar. For example, type:

http://192.168.10.100

2.Type your user name and password in the appropriate fields.
3.Click Log In.
 
If a login failure is caused by a database being down, “- Unable to contact login server. - Please contact Rhythm Engineering support if this problem persists.” displays.
NOTE:Please note that the user name and password fields are case-sensitive.

NOTE:If you have a session timeout associated with your account, you are automatically logged out of In|Sync after the associated inactive time is reached. If you are logged out because of inactivity, re-logging into In|Sync displays the last screen opened before you were logged out.
NOTE:The functionality that is available to you through the WebUI is dependent upon the user access set up for your username. This can be  accessed by an administrator on the Edit Users screen.
