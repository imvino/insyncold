Set the NTP Server

      

For most In|Sync systems, the In|Sync Processors act as peers and automatically synchronize with each other. However, if your network uses a Network Time Protocol (NTP) server to  synchronize the time of day among a set of distributed time servers and clients, you can set which processor is the NTP server by following the instructions below:

1.Select NTP Server from the Settings screen.
2.Type the IP address of the In|Sync Processor being used as the NTP server in the Enter IP field.

NOTE: The IP address of the In|Sync Processor currently being used as an NTP Server displays above the Enter IP field.

3.Click Test to make sure the server is communicating.
4.Click Save. This sets the NTP Server to the In|Sync Processor that contains that IP address.
NOTE:If an intersection is using a secondary In|Sync Processor as a video only processor, you do not need to define the NTP settings on it.
 
Even if the NTP server goes down, the In|Sync Processors still synchronize with each other.
