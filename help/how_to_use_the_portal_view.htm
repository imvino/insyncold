How to use the Portal

      

Before using the Portal, you must define what management groups, intersections, cameras, DIN Relays and Map View links you want listed on the Portal page (see the next section). This is done in the Portal Designer (accessible from the Settings menu). After defining the information on the page, you can then save it to the processor or save it to all processors on the management group. Then you can select Portal from the Management Group menu.
