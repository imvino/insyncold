Configure an In|Sync:Thermal Intersection

      

For In|Sync:Thermal systems, you must complete the following steps to fully configure vehicle detection for each direction:

1.

Add In|Sync detection cameras.

2.

Add detection zones.

3.

Add segment lines.

4.

Define presence segments.

5.

Set fog zones.

6.

Confirm detection cameras are correctly assigned.

7.

Update the intersection.

NOTE:

DO NOT make any changes to the Day-Night Switch Duration or Sunset/Sunrise Offset parameters unless advised to do so by Rhythm Engineering.
