Create a Portal the First Time

      

When you first open the Portal, there are no links. You must first design the Portal in the Portal Designer. To design the Portal:

1.Select Portal Designer from the Settings menu.
2.Type a name for the portal in the field below the Settings heading.
3.You can either try to auto-generate the Portal view by clicking the Auto Generate button or you can manually create the Portal. To manually create the Portal, add management groups, intersections, camera IP addresses, maps and DIN relays.
NOTE:You must know the IP addresses for all the equipment you want to monitor.
