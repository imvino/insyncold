Add a Monitoring Camera

      

NOTE: The following instructions pertain only to In|Sync:Tesla installations.

 

For troubleshooting purposes, Rhythm Engineering requires that a monitoring camera be in place at each intersection. This allows traffic engineers to see what is going on at an intersection remotely and allows Rhythm Engineering personnel to remotely monitor/configure settings. To add a monitoring camera within the Configure utility:
 

1.If the Configuration Utility is not open, select Configure Detectors from the Settings menu. The Configure Detectors screen displays.
2.Click Launch Configuration. The Login dialog box displays.
NOTE:You may be asked whether you want to run the application. Select the Do not show this again for apps from the publisher and location above checkbox and click Run.
3.Type the same user name and password you used to log into the In|Sync WebUI in the Login dialog box. The ConfigApplet dialog box displays.
NOTE:If you see a Security Warning dialog box, click Continue.
4.In the tree view, right-click the intersection’s name.
5.Select Create In|Sync Panomorphic Camera.
6.Type the IP address for the camera in the IP Address field. Leave the user name and password set as the defaults.
7.Click OK.
