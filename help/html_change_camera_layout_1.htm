Change camera layout

The Multiple Camera view provides several different layouts for viewing live images, which can be selected from the Layouts drop-down list box:

▪

Standard Quad: Camera views displays two views horizontally and two views vertically.

▪

Diamond: Camera views are displayed in a diamond shape.

▪

2 Column: Camera views display in two columns.

▪

3 Column: Camera views display in three columns.

▪

4 Column: Camera views display in four columns.
