Managing Users

      

Logging into In|Sync requires you to have a user account. Setting up a user account requires you to have a user name and password, along with access to certain functionalities. This information is set up in the Edit Users screen. From the Edit Users screen, you can:

▪Add a new user account
▪Edit existing user account’s functionality access
▪Change a user’s password
▪Delete a user account  
NOTE:Managing user accounts is a functionality that is available to users that have Administer Users access (set in the Account>Edit Users screen).
NOTE:In|Sync provides two default user accounts upon installation: ADMIN and VIEW. We recommend that you either change each account’s password or delete these accounts entirely and create accounts unique to each user of the In|Sync system.

The following table defines what functionality is available for each user permission setting.

 

User Permission

Functionality

Web/API Access

API Only: Provides the ability to invoke Application Programming Interface (API) methods from a program, but cannot be invoked using a browser.

WebUI and API: Provides access to the API methods from a program and from a browser.

Cameras

Disabled: Cannot view the single or multiple camera views

View Only: Only view the single or multiple camera views, camera status, In|Sync Status, and Processor Status

View/Control: You can view the single camera or multiple camera view AND perform the following functions on the Single Camera View:

▪Reboot Camera
▪Grab Background
▪Toggle Emergency
▪Override Fog Mode
▪Record Video

You can also set up a Recording Schedule.

Session Timeout

Sets the amount of inactive time allowed before the In|Sync WebUI automatically logs the user out of the system.

User Enabled

Able to log in to the In|Sync WebUI

View Reports

Run all three types of reports

In|Sync Maintenance

Restore a configuration, save an archive, or deploy the application from a file.

Management Group Maintenance

If this is the only thing checked, you can view the arterial's LiveMap. If View Cameras is also enabled, you can view the portal or management group view.

Configure Intersection

Gives access to configure detectors, rename phases, re-IP equipment, set GPS coordinates, set NTP server, design the management group view, and design a portal.

Manual Traffic Calls

In order to use this, you must have Cameras set to View Only or View/Control. If Manual Traffic Controls is enabled and Cameras is set to View Only, you can manually control an intersection.

Administer Users

View the Invalid Login report or set up users and each user's security permissions.
