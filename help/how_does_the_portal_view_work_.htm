How does the Portal work?

      

Portal is user-customizable, meaning that you can define exactly what management group, intersection, camera, DIN relay, and live map links show when the Management Group Portal screen displays. This is done through the Settings/Portal Designer screen. After defining which links display in the Portal screen, you can save it to a processor or save it to all processors in the management group so that the view can be accessed when connecting to any processor within the management group.
