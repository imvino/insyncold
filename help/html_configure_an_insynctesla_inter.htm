Configure an In|Sync:Tesla Intersection

      

For In|Sync:Tesla, you must complete the following steps to fully configure vehicle detection for each direction:

1)Add the monitoring camera.
2)Add phases.
3)Assign lanes.
4)Add external detectors.
a)Define detector settings.
a)Add external detectors to lanes.
5)Confirm detectors are assigned correctly.
6)Update the intersection.
