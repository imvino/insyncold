Add a Custom Camera to a Column

      

Instead of displaying all the cameras for an intersection within one column, you may want to display a camera in a new column. To do that, complete the following steps:

1.Add a blank column by selecting Column from the + ADD menu.
2.Give the column a name by replacing "Unnamed Column" with a meaningful name.
3.Click the + Camera button in the new column. The Add Custom Camera dialog box displays.
4.In the Intersection IP field, type the IP address of the processor located on the intersection where the camera resides.
5.Click Update List. A list of Camera Names is updated in the Camera Name drop-down list box.
6.Select the camera you want displayed from the Camera Name drop-down list box.
7.Click Save.
