Open the Configuration in CentralSync

      

For convenience, InSync provides an option for you to open a corridor’s configuration within the CentralSync configuration management software. This allows you to easily tweak a configuration as you are monitoring it. To launch CentralSync from the InSync WebUI:

1.Select CentralSync from the Corridor menu. The CentralSync Launcher dialog box displays.
2.If you do not have CentralSync installed on your computer, this dialog box provides you with a link to download the CentralSync software. If you do have CentralSync installed on your computer, click Continue. The Launch Application dialog box displays.
3.Verify that the Rhythm Tool Version Selector option is highlighted and click OK. The correct version of CentralSync opens and the Corridor Receive - Step 1 of 2 dialog box displays.
NOTE:The InSync WebUI and CentralSync version numbers must match each other in order to launch the CentralSync. For instance, if you are running InSync 1.4, you must have CentralSync 1.4 installed on your computer in order to open the corridor’s configuration up in CentralSync.
4.Type your user name and password to access the intersection’s InSync Processor in the related fields and click OK. CentralSync downloads the configuration files from that processor and displays the progress in the Corridor Receive - Step 2 of 2 dialog box.
5.Click Finish to complete the configuration download process. You can now edit the configurations and then save them. To upload the configuration back to the InSync Processor, select Send to InSync from the File menu.
NOTE:For more information on editing configurations in CentralSync, see the CentralSync User’s Guide.
