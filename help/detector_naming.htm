Detector Naming

      
▪Verify that all detectors in the management group are labeled with meaningful names for identification purposes.

 Accepted symbols include: alphanumeric letters (A through Z; 0 through 9), hash mark (#), and underscores ( _ ). You cannot use hyphens or backslashes.

▪Make sure you give the detector a descriptive name so that you remember where the detector is located; we recommend using the following format:
 
[phase number] _ [lane/subphase number (left to right)] _ [detector order (front to back)]
•6_1_1 (phase 6, lane 1, first detector)
•6_1_2 (phase 6, lane 1, second detector)
•6_2_1 (phase 6, lane 2, first detector)
