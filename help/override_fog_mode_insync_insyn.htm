Override Fog Mode (In|Sync/In|Sync:Fusion Only)

      
NOTE:Overriding Fog Mode is only available to users with administration access.

Selecting this button disables fog mode. It is an override for when the cameras think they have a compromised view, but a human has confirmed that they can see just fine. When you click this button, you disable Fog Mode for that calendar day. It is then automatically re-enabled the next morning. To override fog mode:

1.Select Single Camera View from the Views menu.
2.Choose the camera direction from the Camera drop-down box.
3.Click the Override Fog Mode button.
