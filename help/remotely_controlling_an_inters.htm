Remotely Controlling an Intersection

      

With In|Sync’s live camera views, you can see how In|Sync is responding to real-time traffic demand. You can also, with the correct permission level, manually control an intersection’s traffic signals remotely using the In|Sync WebUI. In addition, you can perform troubleshooting steps remotely to save you a trip to an intersection. The following section details how to use the manual controls that come with the In|Sync WebUI.
